Chabot's Computer Hardware Club

We make things practical!

Our club's primary objective is to help each other learn and discuss about Computer Hardware.



Join our Discord Server here!

Some of the stuff that we do are:

  • Building PCs
  • Computer Safety
  • Software Installation
  • Talking about Hardware-related Topics
  • and more!
